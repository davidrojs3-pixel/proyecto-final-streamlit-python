# App analizadora de datasets con Streamlit

Proyecto final del módulo **Exploración y visualización de datos con Python**.

## Objetivo

Construir una aplicación interactiva en Streamlit que permita cargar, validar, procesar y visualizar cuatro datasets de distintos contextos:

- AI Impact on Jobs 2030
- Superstore
- E-commerce Risk
- Teen Mental Health

## Estructura

```text
proyecto_final_streamlit/
├── app.py
├── requirements.txt
├── README.md
├── data/
│   ├── AI_Impact_on_Jobs_2030.csv
│   ├── sample_-_superstore.csv
│   ├── synthetic_ecommerce_order_risk_dataset.csv
│   └── Teen_Mental_Health_Dataset.csv
└── assets/
```

## Ejecución local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Secciones de la app

1. Home
2. Carga y perfil del dataset
3. Procesamiento de datos
4. Análisis visual

## Uso responsable

Los resultados son exploratorios y no reemplazan validación técnica, estadística, médica, financiera ni profesional.

## Autoría y revisión

El proyecto debe ser revisado, adaptado y probado por el estudiante antes de su entrega final.
